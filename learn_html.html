Introduction to HTMl

HTML or Hyper Text Markup Language is used for creating basic webpage layout. It is used to put up the skeleton, over which styling is done and functionalities are introduced through scripting.

HTML consists of a series of element that tell the browser how to display the content. These elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
